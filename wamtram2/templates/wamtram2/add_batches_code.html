Batch Code Guide

The batch code consists of three parts:

  1. Part 1 (2-3 chars): N followed by the night number (e.g., N1, N2, N10)
  2. Part 2 (2 chars): Location code (see table below)
  3. Part 3 (2 chars): Year (e.g., 24 for 2024)

Example: N1GN24 (Night 1, Gnaraloo, 2024)
